# 🧠 Redes Neurais e Deep Learning

Este repositório documenta minha jornada no aprendizado de arquiteturas neurais profundas. Aqui, exploro desde os fundamentos matemáticos do Perceptron até a implementação de modelos complexos de **Multi-Layer Perceptron (MLP)** utilizando os frameworks líderes da indústria.

## 🚀 Frameworks e Ferramentas
- **Deep Learning:** PyTorch e TensorFlow/Keras.
- **Processamento:** Tensors (tensores) e arrays multidimensionais (NumPy).
- **Visão Computacional:** Dataset MNIST (reconhecimento de dígitos manuscritos).

---

## 📂 Conteúdo Técnico

### 1. Fundamentos e Álgebra de Tensores
- **Manipulação de Tensores:** Criação e operações com tensores no PyTorch, base essencial para o fluxo de dados em redes neurais.
- **Lógica do Perceptron:** Implementação manual de cálculos de pesos, bias e funções de ativação (como **ReLU**) para entender como a informação flui através de um neurônio artificial.

### 2. Implementações de MLP (Multi-Layer Perceptron)
Demonstração de competência técnica em diferentes ecossistemas para resolver o problema clássico do MNIST:
- **Abordagem PyTorch:** Uso de `torch.nn`, `optim` e `DataLoader` para construir um ciclo de treinamento personalizado e eficiente.
- **Abordagem TensorFlow/Keras:** Implementação via API Sequencial, destacando a facilidade de modelagem de camadas e compilação de modelos de classificação.
